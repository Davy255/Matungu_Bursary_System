Registration Status

Toggle this to enable or disable user registration and application submissions system-wide.

Deadline Setting

Set a specific date and time for when registration should automatically close. After this deadline passes, new registrations and application submissions will be disabled automatically.

Important Notes
  • Changes take effect immediately
  • Users cannot register when registration is closed
  • Users cannot submit new applications when registration is closed
  • Existing draft applications can still be edited
  • System checks deadline status on every registration/submission attempt
⚠ Warning: Once registration is closed, notify applicants through the notification system.
