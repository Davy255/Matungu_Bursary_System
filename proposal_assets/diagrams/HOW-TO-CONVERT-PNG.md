# SVG to PNG Conversion Guide

## ❓ Why SVG saves as SVG (not PNG)?

When you right-click an SVG file in a browser, it saves as SVG because **that's the actual file format**. SVG is a vector format, not a raster image like PNG.

---

## ✅ BEST METHOD: Windows Snipping Tool (Built-in, No Installation)

### For EACH diagram:

1. **Open the SVG file** in your browser (double-click it)

2. **Press:** `Win + Shift + S` on your keyboard
   - Your screen will dim slightly
   - A toolbar appears at the top

3. **Click "Rectangular Snip"** (first icon in toolbar)

4. **Drag to select** the entire diagram area

5. The screenshot is **automatically copied** to your clipboard

6. **Open Paint:**
   - Press `Win + R`
   - Type: `mspaint`
   - Press Enter

7. **Paste:** Press `Ctrl + V`

8. **Save as PNG:**
   - File → Save As → PNG image
   - Name it properly (e.g., "database-schema.png")
   - Choose your location

9. **Repeat** for all 5 diagrams

## ✅ Where to Use PNG Files

**Microsoft Word (Proposal):**
- Insert → Pictures → Select PNG file
- Resize as needed (high quality maintained)

**PowerPoint (Presentation):**
- Insert → Pictures → Select PNG file
- Images will be crystal clear on projection

**Printing:**
- PNG files print perfectly at any size
- No pixelation or blurriness

---

## 🔍 Quality Check

After converting, your PNG files should be:
- ✅ Clear and readable
- ✅ Around 1920x1600 pixels (or larger)
- ✅ File size: 500KB - 2MB each
- ✅ 300 DPI equivalent (print quality)

## 🆘 Still Having Issues?

**If Snipping Tool doesn't work:**
1. Press `Win + PrtScn` (takes full screenshot)
2. Screenshots save to: `Pictures\Screenshots\`
3. Open in Paint and crop to diagram area
4. Save as PNG

**If online converters don't work:**
1. Open Microsoft Edge/Chrome
2. Open SVG file in browser
3. Right-click → "Print" → Save as PDF
4. Then convert PDF to PNG using: https://pdf2png.com/
